## 🔋 电池健康（6个 Dashboard）

### 核心健康度
- **电池健康度** - 电池容量衰减、健康度百分比
- **续航曲线图** - 实际续航趋势、与表显对比
- **电池容量曲线图** - 容量变化趋势

### 充电分析
- **充电曲线统计** - 充电速度曲线、峰值功率
- **快充曲线图(按运营商)** - 对比不同充电桩速度
- **充电费用统计** - 费用分析、成本对比

**适用场景**：
- 怀疑电池衰减 → 看「电池健康度」
- 续航不准 → 看「续航曲线图」
- 充电慢 → 看「充电曲线统计」

---

## ⚡ 充电管理（5个 Dashboard）

### 实时充电
- **当前充电状态** - 实时功率、电压、电流、预计时间
- **充电记录** - 每次充电详情、电量变化

### 统计分析
- **充电统计（汇总）** - 充电次数、总电量、平均功率
- **充电费用统计** - 费用明细、月度对比、成本分析
- **充电费用统计** - 按地点分析、运营商对比

**适用场景**：
- 正在充电 → 看「当前充电状态」
- 查历史记录 → 看「充电记录」
- 算花了多少钱 → 看「充电费用统计」

---

## 🚗 驾驶分析（7个 Dashboard）

### 行程总览
- **行程统计（汇总）** - 总里程、总次数、平均距离
- **行程统计（年月日）** - 按时间维度分析
- **行程统计（时间段）** - 自定义时间范围统计
- **行程统计（每次充电）** - 单次充电间隔行程

### 行程详情
- **行程列表** - 每次行程明细、起点终点
- **驾驶记录追踪** - 地图轨迹、详细数据
- **最近车速统计** - 速度分布、驾驶习惯

**适用场景**：
- 看跑了多远 → 「行程统计」
- 看具体路线 → 「驾驶记录追踪」
- 分析驾驶习惯 → 「最近车速统计」

## 📍 位置服务（4个 Dashboard）

### 地点记录
- **访问过的地点** - 常去的地方、停留时间
- **足迹地图** - 可视化行驶轨迹

### 地理围栏
- **当前车辆状态** - 显示当前位置
- **充电费用统计** - 按地点分析充电费用

**适用场景**：
- 看去过哪里 → 「足迹地图」
- 找常去的地方 → 「访问过的地点」
- 看当前位置 → 「当前车辆状态」

## 🏠 车辆状态（6个 Dashboard）

### 实时状态
- **当前车辆状态** - 电量、续航、状态、位置
- **当前充电状态** - 充电实时监控
- **当前驾驶状态** - 驾驶实时监控

### 历史状态
- **最近车辆状态** - 最近记录的时间线
- **状态** - 在线/离线/休眠统计
- **时间线** - 状态变化事件流

**适用场景**：
- 看现在怎么样 → 「当前车辆状态」
- 查昨晚状态 → 「最近车辆状态」
- 看在线时长 → 「状态」
